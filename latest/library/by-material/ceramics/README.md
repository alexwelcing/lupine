# Ceramics & Minerals Collection

## Datasets

### Gallium Nitride (GaN)
- **CN-NBSDC-001**: Multi-physics removal model
  - Nano-scratch simulations
  - Hard-brittle material behavior
  - Part of 55.85GB cross-scale dataset

### Silicon Carbide (SiC)
- **CN-NBSDC-001**: Multi-physics removal model
  - Abrasive machining simulations
  - Subsurface damage analysis
  - SEM/TEM validation data

### Lithium Tantalate (LiTaO3)
- **CN-NBSDC-001**: Multi-physics removal model
  - Piezoelectric material focus
  - [10-10], [0001] orientations

### Magnesium Aluminate Spinel (MgAl2O4)
- **CN-NBSDC-001**: Multi-physics removal model
  - Transparent armor applications
  - High hardness ceramic

### Calcium Carbonate (CaCO3)
- **WEST-MC-002**: Phase transitions (11.8MB)
  - Raiteri potential
  - Temperature-pressure phase map
  - Aragonite-calcite transitions

## Common Properties
- **Bonding**: Ionic/covalent mixed
- **Simulation challenges**: Brittle fracture, phase transitions
- **Validation**: Often paired with nanoindentation experiments
- **Typical potentials**: ReaxFF, Buckingham, custom

## Viewer Notes
- Large bandgap materials (transparent)
- Defect visualization requires stress/strain coloring
- Phase transitions benefit from animation
