Six Meta-Scientific Hypotheses

The old hypotheses (Simpson's paradox, hyper-ribbon dimensionality) are abandoned. We formulate new hypotheses about the epistemic structure of validation itself — all formally stated, all falsifiable, none yet proven.

H1 — [CONJECTURE]

Validation Incompleteness

For any finite benchmark B and potential family P, there exists an observable outside B whose error is unbounded. No finite benchmark exhausts a many-body interaction law.

H2 — [CONJECTURE]

Epistemic Entropy Bound

Error entropy is bounded below by Kolmogorov complexity divided by benchmark size. Simpler potentials concentrate error; complex potentials disperse it.

H3 — [CONJECTURE]

Causal Structure Theorem

In the true causal graph, crystal structure is a mediator, not a confounder. Simpson's paradox cannot arise from elemental stratification in real data.

H4 — [CONJECTURE]

Spectral Rigidity

Eigenvalue multiplicities of the error covariance matrix equal irreducible representation dimensions of the crystal's point group. PR is determined by symmetry.

H5 — [CONJECTURE]

Transferability Phase Transition

There exists a critical parameter count P_c such that P < P_c yields correlated errors (low PR) and P > P_c yields decorrelated errors (PR → N).

H6 — [CONJECTURE]

Bootstrap Collapse

For N < 30 validation points, the 95% bootstrap CI on PR has width > N/2 with probability > 0.5. Small samples make PR claims statistically void.
