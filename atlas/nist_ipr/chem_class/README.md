## File-tracking convention

Each potential's parameter files are listed in `manifest.json` under `entries[*].potentials[*].artifacts[]` with both the upstream `https://www.ctcms.nist.gov/...` URL and the local mirror path `atlas/nist_ipr/files/<pair_style>/<id>/<filename>`.

To populate the local mirror, run `python scripts/nist_ipr_mirror.py atlas/nist_ipr` from the repo root.
