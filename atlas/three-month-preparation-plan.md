# ATLAS Three-Month Preparation Plan
## Building Optionality for Summer Decision

**Current Date:** March 2026  
**Target Decision Window:** June-August 2026  
**Status:** Preparing for strategic raise with US/China optionality

## Executive Summary

You have 12 weeks to transform from "researcher with code" to "investable entity with optionality." The goal isn't to raise now — it's to be *ready* to raise, with both sides cultivated, so you can run a proper process in summer.

**Three parallel workstreams:**
1. **Personal foundation** (passport, residency, family logistics)
2. **Business foundation** (entity, IP, documentation, team)
3. **Relationship foundation** (warm intros, intelligence gathering, soft cultivation)

|--------|------|------|----------|
| **Stay US** | Easy, family stable | Limits China access, optics | Immediate |

**Month 1 actions:**

### Month 2: Family Logistics (Weeks 5-8)

- [ ] Costs: $20-40K/year per child for top tier

- [ ] Set up registered address
- [ ] Open corporate bank account
- [ ] Cost: S$2-5K setup, S$3-5K/year ongoing

**Corporate structure:**
```
ATLAS Labs Pte Ltd (Singapore)
├── Owns: Core IP, patents, trademarks
├── Licenses to: Operating subsidiaries
└── You: Founder, shareholder, potentially EP holder
```

**IP assignment:**
- [ ] Document all existing code contributions
- [ ] Assign all IP to Singapore entity
- [ ] Ensure clean chain of title
- [ ] Cost: Legal fees $5-15K

**Export control determination:**
- [ ] **CRITICAL:** File Commodity Jurisdiction (CJ) request with BIS
- [ ] Determine if ATLAS is EAR99 (not controlled) or specific ECCN
- [ ] Timeline: 14 days (statutory), reality 30-60 days
- [ ] Cost: $10-25K with export counsel
- [ ] **Do this now** — affects everything else

**Trademark protection:**
- [ ] File "ATLAS" trademark in US, Singapore, China
- [ ] Consider: ATLAS VIEW, ATLAS SIM, etc.
- [ ] Cost: $5-10K per jurisdiction

## The Decision Framework (June)

At the end of 3 months, you'll face a choice:

**Option A: Run dual-track auction**
- Formal process with both sides
- Compressed timeline (4-6 weeks)
- Winner-take-all outcome
- Maximum valuation potential
- Risk: Both sides walk

**Option B: Choose one side now**
- Single-track raise with preferred partner
- Stronger relationships likely
- Simpler process
- Lower valuation ceiling
- Risk: Missing better option

**Option C: Delay, continue building**
- Not ready, need more time
- Bootstrap or small bridge
- Raise in fall/winter
- Risk: Missing current window

**Decision criteria:**
- Strength of relationships on each side
- Market timing/geopolitical climate
- Personal/family readiness
- Product milestones achieved
- Competitive landscape shifts
