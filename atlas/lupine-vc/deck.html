Differentiator

The DFT → ML → MD pipeline
is the killer feature

No existing tool covers the full pipeline. Lupine provides a single-command workflow from quantum calculation to production-scale simulation.

# One command: DFT training data → ML potential → validated model
lupine flow train-mlip \
  --structure POSCAR \
  --model-type mace \
  --target-delta 0.5  # meV/atom accuracy

# Then: production MD at near-DFT accuracy
lupine md -i production.lmp  # LAMMPS format, 1M steps
